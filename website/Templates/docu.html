Software setup

The elaborated C++ code for controlling RoboFace has beautiful user frienly python binaries. The following code example speaks for itself: 


	>>> import face
	>>> f = face.Face()
	>>> f.sad()                 # sad facial expression with head movement
	>>> f.sad(False)            # sad facial expression with no head movement
	>>> f.happy()               # happy facial expression with optional head movement
	>>> f.unsure()              # unsure facial expression with optional head movement
	>>> f.angry()               # angry facial expression with optional head movement
	>>> f.neutral()             # neutral facial expression with optional head movement
	>>> f.moveHead(120, 160)    # move head horizontally and vertically
	>>> f.moveHeadX(120, 160)   # move head horizontally
	>>> f.moveHeadY(160)        # move head vertically

For using the face attribute detection software, a standard python (2.7 or 3.5) anaconda installation should additionally include opencv, keras. Additionally, for text to speech synthesis, you should install gTTS, pygame. You can easily install them using conda with the following commands: 


	conda install -c menpo opencv3=3.1.0
	conda install keras
	pip install gTTS
	pip install pygame

Then for running the software, execute the python script face_detection/face_detection.py

If you eventually want to retrain the network for further improvement of the results, then you should take a look in te neural_network_training/train_normalised.py script.
